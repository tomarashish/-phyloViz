phyloGraph.js


phyloGraph is a D3.js based JavaScript library for created phylogenetic tree visualization. It makes easy to create phylogenetic tree on web pages and allows to integrate genomics metadata.

Features

Tree visualization along with metadata.
phyloGraph is a reusable d3.js module which creates a resizable SVG container.

CONTROL OPTIONS

  • Add/remove link weight
  • Add/remove link values display
  • Add/remove link name display
  • Add/remove node size
  • Chnage tree display (linear/radial)
  • Link extension

ADD METADATA

  • Add rectangle
  • Add bar chart
  • Add pie chart
  • Add alignment sequence
